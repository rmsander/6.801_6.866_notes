# 6.801_6.866_notes
A repository containing notes and LaTeX for MIT's 6.801/6.866 Machine Vision course.  This course is taught by Professor Berthold Horn, and focuses on geometric computer vision.

To view the PDF lecture notes, simply select the lecture number from this GitHub repository.  To edit the LaTeX code used to generate these PDFs, you can do so by editing the files in `tex`.
